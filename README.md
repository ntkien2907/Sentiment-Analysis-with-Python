## Sentiment Analysis


### How to use
* The instruction below is done on Windows 10.
* Step 1: Automatically install all dependencies from `requirements.txt`.
    ```
    pip install -r requirements.txt
    ```
* Step 2: Configure hyper-parameters in `config.py`.
* Step 3: Start server.
    ```
    python app.py
    ```
